# 웹 기반 게임 추천 서비스

## 프로젝트 소개
해당 프로젝트는 게임이라는 장르에 익숙하지 않은 분들에게 게임을 추천해주고 그에 대한 정보를 제공하는 웹 페이지입니다. 메인 페이지에서 간단한 설문 조사를 통해 게임을 추천받고 회원가입을 통해 게시판 이용, 다른 이용자들과 소통할 수 있습니다
